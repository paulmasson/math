am( z, m )

The Jacobi amplitude function of z and parameter m in Math. Defined as the inverse of the incomplete elliptic integral of the first kind:

\[ z = \int_0^\phi \frac{ d \theta }{ \sqrt{ 1 - m \sin^2 \theta } } \qquad \rightarrow \qquad \operatorname{am}( z | m ) = \phi \]

Note that all Jacobi elliptic functions in Math use the parameter rather than the elliptic modulus k, which is related to the parameter by \( m = k^2 \).
