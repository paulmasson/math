cn( z, m )

The Jacobi elliptic cosine of z and parameter m in Math. Defined as the cosine of the Jacobi amplitude:

\[ \operatorname{cn}( z | m ) = \cos [ \operatorname{am}( z | m ) ] \]

Note that all Jacobi elliptic functions in Math use the parameter rather than the elliptic modulus k, which is related to the parameter by \( m = k^2 \).
